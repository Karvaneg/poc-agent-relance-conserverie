# Skills — Index

Ce dossier contient les skills et fichiers de contexte de Claude Code pour ce projet.

Pour créer un nouveau skill local à ce projet, ajoute un fichier `<nom>.md` dans ce dossier (`.claude/skills/`) avec un frontmatter YAML (`name` + `description`). Tu peux partir du fichier `_template.md` fourni ici. Il sera détecté automatiquement à la prochaine session Claude Code.

---

## Contenu

| Fichier | Rôle | Invocation |
|---------|------|------------|
| project-context.md | Contexte projet auto-généré | (généré, ne pas éditer) |
| _template.md | Squelette à dupliquer pour créer un nouveau skill local | (copier-coller) |
